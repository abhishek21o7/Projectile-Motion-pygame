# Projectile Motion Simulator (Python + Pygame)

A physics-based projectile motion simulator built using Python and Pygame.  
This project visualizes projectile trajectory and calculates important parameters like range, maximum height, and time of flight.

## Features

- Real-time projectile animation
- Trajectory path visualization
- Range calculation
- Maximum height calculation
- Time of flight display
- Grid and axis visualization
- User input for velocity and angle

## Physics Concepts Used

- Projectile motion equations
- Horizontal and vertical velocity components
- Gravity acceleration

Equations:

Range:
R = v² sin(2θ) / g

Max Height:
H = v² sin²θ / (2g)

## Requirements

Install pygame:
>>pip install pygame


## Controls

- Enter velocity
- Enter angle
- Press SPACE to launch
- Press "R" to reset
